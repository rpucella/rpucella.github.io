Programming Languages (Spring 2017)

CS 152 / Tue-Thu 09:00-10:30 / Maxwell Dworkin G125

This course is an introduction to the theory, design, and implementation of programming languages. Topics covered in this course include: formal semantics of programming languages (operational, axiomatic, denotational, and translational), type systems, higher-order functions and lambda calculus, laziness, continuations, monads, as well as formal reasoning about programs using proof assistants.

Course Information

Prerequisites: CS 51. CS 121 is recommended. (1) this is not an introduction to programming; you should already know how to program, ideally in at least couple of languages; (2) we'll be doing math — you must be very comfortable with recursion, proofs, basic mathematical ideas and notations, including sets, relations, functions, and especially induction.

Instructor: Riccardo Pucella (rpucella@seas.harvard.edu)

Office hours: Tue 10:30-11:30 / MD 233 (Prof. Chen's office)

Textbooks: There is no required textbook for the course. We will be working off class notes and online references.

While there is no required textbook, there are excellent texts about the topic of the course, which you should definitely seek out if you feel you need the extra help:

  • J. C. Mitchell, Foundations for Programming Languages, Cambridge University Press, 2003.
    Available online through Harvard University Libraries eContent Collection. To view on a Mac, follow these instructions.
  • R. Harper, Practical Foundations for Programming Languages, Cambridge University Press, 2016.
  • B. C. Pierce, Types and Programming Languages, MIT Press, 2002.
    Available on reserve at the library.
  • G. Winskel, The Formal Semantics of Programming Languages: An Introduction, MIT Press, 1993.
    Available on reserve at the library.
  • S. Krishnamurthi, Programming Languages: Application and Interpretation
    Available from the author's web page.

Grading: There will be an in-class midterm and a final exam. There will be 6 homework assignments. Your grade will be determined by a weighted average of your scores on homework assignments (50%), the midterm exam (20%), the final exam (25%), and class participation (5%). The weights are meant to be indicative and are subject to change. Participation includes attendance and participation in class, section, and office hours, and contributing to online discussion.

We have some policies on lateness and collaboration.
