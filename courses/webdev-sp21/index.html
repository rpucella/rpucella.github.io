Web Development (Spring 2021)

ENGR 3599 / TBA / Online

An intensive introduction to the full stack of modern web development — topics I would like to cover include:

  • HTML and CSS
  • JavaScript
  • TypeScript
  • Client-side frameworks (React, Angular, Vue, ...)
  • Server-side rendering (templates)
  • Server-side frameworks (Flask, Express, ...)
  • Persistence
  • Websockets
  • Deployment (local, cloud)

The structure of the course will be pretty straightforward: we'll start with writing code purely client-side (on the browser). Then we'll introduce servers, and see what role they play and how they support what occurs client-side. Then we'll introduce a persistence layer (database) to maintain information long-term between server runs or even shared between servers.

Client-side code will probably be in TypeScript using React, and server-side code will probably be in Python using Flask. Everything we do will carry over to other frameworks, but we have to choose something.

Course work will probably take the form of a sequence of homeworks building a complete working web application from scratch. There will also be a project, which is going to be yet another web application, this time of your own design.
